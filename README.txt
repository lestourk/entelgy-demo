Tiene 2 endpoints de metodo POST

* Con parametros de entrada (todas son cualquier entero positivo)
URL: http://localhost:8080/entelgy/commentsConArgs
Request:
{
    "cantidad": 10,
    "llaveId": 1000,
    "llaveComentario": 2000
}

* Sin parametros de entrada
URL: http://localhost:8080/entelgy/commentsSinArgs
